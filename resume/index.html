(X)HTML, HTML5

生涯伊始，在写过了很多 HTML 页面之后发现有什么不对劲，大多标签都要手写两次，Dont Repeat Yourself 的话一直重复，对于懒人的我来说很适时的遇到了 HTML Zencoding，这真是个法宝，不仅加快了 Coding 速度，也让我更理解了页面结构。之后在接触了 Node.js 后让我有幸得知了模板语言 Jade，我就只能这么说：“That's What I Wanted! Fucking Amazing!”
CSS2.1, CSS3

一直以来，书写 CSS 的方式与顺序总是这样的，重置浏览器默认样式 &raquo; 定义组件 &raquo; 按页面结构进行书写。最后一个很大很大的 master.css 诞生了。但 CSS 也可以很 DRY(Dont Repeat Yourself)。Stylus - Dynamic CSS Language. 使 CSS 更“干燥”，不仅一定程度上简化了 CSS 的语法，在写基本样式定义、 CSS3 浏览器兼容语法等时更得心应手。辅以 Modrenizr，HTML5 Boilerplate 来处理向前与向后兼容。
JavaScript

对于思维偏向设计的我，JavaScript 一直是我的短板，不得要领。也学习使用过很多框架，jQuery 算是用的最多的，没有进行过整站式的 Module 编写，多者只是简单的事件绑定，仍需下很大功夫！接触了 Node 之后，对 HTML, CSS, JavaScript 都有了更进一步的理解，有热爱的开源社区 Github 为良师益友，进度不会慢！
